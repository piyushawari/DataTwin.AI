## About The Project  

In modern AI systems, accessing real-world data is challenging due to privacy concerns and regulations. This creates a gap between data availability and data security.

DataTwin AI is a privacy-focused synthetic data generation system that creates artificial datasets similar to real data while ensuring no real user information is exposed.

The system uses generative modeling techniques to learn patterns and relationships in data and generate new datasets with similar statistical properties.

---

##  Key Features  

- Synthetic Data Generation  
- Privacy Protection (no real data leakage)  
- Interactive Dashboard (Streamlit)  
- Data Validation using graphs  
- Machine Learning Utility Check

## How It Works  

1. Upload dataset (CSV)  
2. Train generative model  
3. Generate synthetic data  
4. Compare real vs synthetic data  
5. Evaluate performance
